# MBTI診断サイト

MBTI（16タイプ性格診断）テストを提供するWebアプリケーションです。

## 機能

- MBTI診断テスト（20問）
- 結果のグラフ表示
- SNS共有機能
- レスポンシブデザイン

## 技術スタック

- React
- TypeScript
- Tailwind CSS
- Chart.js
- React Router

## セットアップ

1. リポジトリをクローン
```bash
git clone [repository-url]
cd mbti
```

2. 依存関係のインストール
```bash
npm install
```

3. 開発サーバーの起動
```bash
npm start
```

4. ブラウザで http://localhost:3000 を開く

## ビルド

本番用ビルドを作成する場合：

```bash
npm run build
```
